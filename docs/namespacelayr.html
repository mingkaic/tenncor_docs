PybindT  discount_factor = 0.99,
PybindT  epsilon = std::numeric_limits<PybindT>::epsilon(),
std::string  root_label = "" 
)

Return all batches of variable assignments of momentum-based rms error approximation applied to particular variables-error associations

Momentum-based Root Mean Square Approximation for each (x, err) in leaves momentum_next ~ * momentum_cur + (1 - ) * err ^ 2 x_next ~ x_curr - ( * err) / (sqrt( + momentum_next))

where is the learning rate, is epsilon, and is discount_factor initial momentum is 1
